Chapter 1. 문장의 기초

UNIT 03. There is/are
There is a person
There are people		 ~이(가) 있다
There is no good restaurants near here/nearby
There are no good restaurants near here/nearby		 ~이(가) 없다
There is a lot of time
There are a lot of people		 ~이(가) 많다
There isn’t a lot of time
There aren’t a lot of people		 ~이(가) 많이 없다
